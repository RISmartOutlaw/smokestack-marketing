Frequently Asked Questions

How is this different from Shopify/Square?

Backroom is API-first and built specifically for alternative retail inventory management. Shopify and Square are generic POS systems. We're 10x cheaper ($29 vs $99+/month), faster to set up (5 min vs days), and you own your data.

Can I export my data?

Yes, completely. Export all products, orders, inventory data anytime in CSV format. You can switch providers, self-host, or integrate elsewhere. Zero lock-in.

What if I only have 50 orders/month?

Use the Free tier forever. It includes 100 orders/month, so you're covered. Only upgrade when you grow past 100 orders/month.

Can I use this with Shopify?

Yes! Backroom integrates with Shopify via webhook. Create order on Shopify → Backroom auto-deducts inventory → your stock stays in sync. Setup takes 10 minutes.

What about security?

API key authentication, SQL injection protected (ORM), HTTPS everywhere, encrypted database, daily backups. Enterprise-grade security at an indie price.

Can I host this myself?

Yes! Backroom can deploy on your own servers (Railway, DigitalOcean, AWS, etc.). Some owners prefer full control.

What about age verification / compliance?

Backroom tracks inventory; compliance stays your responsibility. Tell us in the intake if you sell nicotine/tobacco and we'll wire the 21+ age gate and warnings into your storefront.

Can I cancel anytime?

Absolutely. No contract. No long-term commitment. Month-to-month only. Cancel anytime. Your data stays yours.

How do I integrate with my POS system?

Backroom has 17 REST API endpoints. If your POS supports webhooks or HTTP requests, it can talk to Backroom. We have integration guides for popular systems.
